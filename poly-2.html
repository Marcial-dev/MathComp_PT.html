Mathematical Problem 1:
•Find the roots of the equation: x^3 + 4x^2 - 4x - 16 = 0
Solution:
(x^3 + 4x^2) + (-4x - 16)
x^2(x+4) -4(x + 4)
(x + 4)(x^2 - 4)
(x + 4) (x - 2) (x+2)
x + 4= 0
x = -4
x + 2 = 0
x = -2
x - 2 = 0
x = 2
 • Roots = {-4, -2, 2}
Mathematical Problem 2:
• Jamie is a Gravity Falls enthusiast and theorist; he finds the equation x^3 + 9x^2 - 9x - 81 = 0 in a hidden part of the Book of Bill. He recalls Alex Hirsch mentioning “roots” once for a secret code for a website. Find the roots of the equation x^3 + 9x^2 - 9x - 81 = 0 to find the secret code for the website. 		Solution:
(x^3 + 9x^2) + (-9x + 81)
x^2(x + 9) + 9(x + 9)
(x + 9)(x^2 - 9)
(x + 9) (x - 3) (x + 3)
x + 9 = 0
x = -9
x - 3 = 0
x = 3
x + 3 = 0
x = -3
 • {-9, 3, -3}
